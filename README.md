#  Garden Tools E-Commerce Website

A responsive e-commerce website for selling gardening tools, built with HTML, CSS, and JavaScript.

##  How It Works
Users can browse a catalog of gardening tools, view product details,
and add items to their shopping cart.

## ✨ Features
- Product catalog with images
- Shopping cart functionality
- Responsive design (mobile-friendly)
- Clean and modern UI
- 
## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON (for product data)
